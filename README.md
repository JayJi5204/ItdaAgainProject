## 프로젝트소개

Itda의 홈페이지를 소개합니다

## 설치 가이드
- IntelliJ 
- Spring Boot 3.x
- Java 17
- MySQL

**초기화**  

    $ git clone https://github.com/JayJi5204/ItdaAgainProject.git 
    $ cd ItdaAgainProject

**실행방법**
1. MySQL 설치  
tt[MySQL 다운로드](https://dev.mysql.com/downloads/windows/installer/)  
2. MySQL 설정  
3. MySQL 실행  
4. IntelliJ 설치  
dd[IntelliJ 다운로드](https://www.jetbrains.com/ko-kr/idea/download/?section=windows)  
5. 프로젝트 Import  
6. 실행
