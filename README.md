# ItdaAgainProject

> **개발기간: 2024.01.02 ~ 2024.00.00**

## 프로젝트소개

Itda의 홈페이지를 소개합니다

## 설치 가이드
- IntelliJ 
- Spring Boot 3.x
- Java 17
- MySQL

**초기화**  

    $ git clone https://github.com/JayJi5204/ItdaAgainProject.git 
    $ cd ItdaAgainProject
