김상준

이번 프로젝트에서 스프링 시큐리티의 인증/인가로 보안을 구현하고자 했습니다. 특히, Role 부여와 인증된 객체를 가져오기 위해 커스터마이징을하면서 심도 있는 이해를 위해 공식 홈페이지를 참고하였습니다.

3월 SQLD 시험을 준비중이며, 개인 프로젝트로 지속적인 서버 운영을 위해 Github Action으로 CI/CD를 배포 자동화를 구축하고자 합니다.

김상준의 깃허브

지성현

이번 프로젝트를 통해서 저는 협업과 소통의 중요성에 대해 좀 더 깨닫게 되었습니다. 개발을 하면서 제가 오류가 생긴 부분을 팀원들이 같이 코드를 보며 오류를 잡았고, 소통을 통해 생각하지 못한 부분에 대해 영감을 얻을 수 있었고, 부족한 점에 대해 피드백을 받아 부족한 부분을 고칠수 있었습니다.

현재는 3월 달에 SQLD 자격증을 취득하는 것을 목표로 공부하고 있고, 5월에는 정보처리기사 자격증을 취득할 계획입니다.

깃허브 주소 :

김현승

이번 프로젝트를 하면서 전에는 하지 못한 게시판을 경험해 보았다. 이번에는 게시판을 만드는 것도 배우긴 했지만 그 외적으로 새로운 개발환경인 맥 , 깃이나 새로운 데이터 베이스인 mysql도 배워보면서 새로운 경험을 해본거 같다.

깃 같은 경우는 전에는 클론으로 받아서 사용했다면 이번에는 직접 깃 레파지토리를 지우고, 백업으로 올릴 깃 레파지토리에 연결을 한 뒤 올리는 작업과 직접 풀 리퀘스트를 해보면서 깃 이그노어의 필요성과 사용하는 방법을 배워서 좋았다

아쉬운것은 시간이 부족해서 댓글기능과 조회수 기능 등을 하지 못한게 굉장히 아쉽다. 또한 게시판에서 유저정보를 가지고 오는것도 굉장히 힘들었다 다음 프로젝트에서는 이 단점을 보완해서 진행하고 싶다.
