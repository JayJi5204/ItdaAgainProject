저희 IT DA는 2023년 2월 8일부터 2023년 9월 18일까지 성남 그린 컴퓨터 아카데미에서 진행된 프로젝트 기반 풀스텍(Frontend,Backend) 개발자(Java,Spring,React)양성, KDT 3기의 1조 팀 이름을 말합니다. IT DA의 의미는 IT를 의미하는 동시에 우리말 '잇다'로 IT와 사람, 플랫폼과 고객을 이어주는 웹페이지를 개발하자를 의미합니다.

KDT 프로젝트를 진행하면서 자신이 부족하다고 느꼈던 부분과 좀 더 공부해보고 싶은 분야의 실력을 키우기 위해 다시 팀을 결성하였습니다. 이번 프로젝트는 Spring Boot와 JPA, Git Branch 전략에 중점을 두었고 만든 프로젝트의 결과물을 AWS를 이용해서 배포하는 것에 목표를 두었습니다.

현재의 사이트가 부족하거나 마음에 들지 않는 점이 많이 있지만 주니어 개발자들이 오늘보다 발전된 내일을 바라며 노력한 결과물이라고 생각해주셨으면 좋겠습니다. 버그나 오류를 찾았을 경우 게시판을 통해서 작성해주시면 빠른 시일내에 버그 및 오류를 수정해 드리겠습니다.
